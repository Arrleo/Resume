# My Static Portfolio 

Персональный сайт-резюме, созданный с использованием чистого HTML5 и CSS3.
**Важно:** данный проект является учебной работой и демонстрацией навыков верстки, а не моим актуальным профессиональным резюме.

Проект демонстрирует навыки адаптивной верстки, работы с формами и стилизации таблиц.

### Структура проекта

Проект имеет максимально простую и легкую структуру, что обеспечивает быструю загрузку страницы:

```text id="prt44x"
Portfolio/
├── index.html    # Разметка страницы (контент, структура, формы)
├── style.css     # Стилизация (градиенты, типографика, адаптивность)
└── photo.jpg     # Фотография профиля
```

---

### Особенности реализации

* **Modern Design:** Использование линейных градиентов и мягких теней для создания «карточного» интерфейса
* **Interactive UI:** Стилизованные поля ввода с эффектами `:focus` и анимацией кнопок
* **Data Presentation:** Таблицы для отображения навыков и опыта
* **Responsive Elements:** Скругление изображений (`border-radius`) и адаптивные контейнеры
* **Contact Form:** Форма обратной связи (демонстрационная реализация)
